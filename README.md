# SOBOK-Server
소상공인 카페 활성화를 위한 카페 투어 생성 및 스탬프 적립 서비스 '소복'의 서버 리포지토리입니다.<br>
'소복'은 숙명여대 캠퍼스타운 스타트업 MVP 서포터즈 '잇다' 활동의 일환으로 제작되었습니다.<br><br>
`서비스 URL` : https://sobok.co.kr/<br><br>
